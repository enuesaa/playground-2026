# Remotion

- react で動画を作れる
- これはすごい。
- `remotion studio` で、動画編集スタジオが立ち上がる
- `remotion render` で、mp4 ファイルが作成される
  - mp4 ファイルが作成される感覚は新鮮。
- `remotion bundle` で何ができるのか不明
  - なんかJSがバンドルされたけど、でもこれをコンテンツ配信？することあるのかな？
- 仕組みをわかってないが各種言語に対応してるらしい
  - ruby, php, go など
  - wasm を使ってそうな雰囲気がある
- ライセンス注意
  - ぱっと見た感じ個人利用はFree License になるっぽい
  - 商用利用は注意が必要そう
- エンジニアがサクッと動画を作って欲しいと言われた時これが手っ取り早いと思った。

## Commands

```bash
npm i
npm run dev
npm run render # render video
```
