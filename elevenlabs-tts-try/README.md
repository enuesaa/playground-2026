# Elevenlabs

- `リアルな音声AIプラットフォーム`
- 実際リアルだった。
- どこよりも品質が高い。実際の人の声に聞こえる
- たまに読み間違いがある程度
- 記号みたいなので話し方を指定できるっぽい
  例: `[早口で] こんにちは`
- モデルはいくつかある様子
  - eleven_flash_v2_5
  - eleven_multilingual_v2
  - eleven_multilingual_v2 の方が後発で高いが、日本語の発音に関していうと eleven_flash_v2_5 の方が自然だった
- JS/Python SDK があり呼び出せる
