# perryts

- ts のネイティブコンパイラらしい
- 手元のコードはコンパイルできなかったけど、example-code はコンパイルできた
  - なんか書きかた悪かったんだろうなあ。
  - mainがどうたらというエラー文言がでた。
  - mainが必須なのかな
- コンパイルしてみるとrustっぽさがすごい
  - エラー文言とかヘルプテキストとか。
  - clapのデフォルトのやつだろうなあ。

## Commands
```bash
# install
brew install PerryTS/perry/perry

# compile
perry compile src/index.ts
```
