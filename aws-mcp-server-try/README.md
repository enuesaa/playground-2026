# AWS MCP Server

- aws cloud の mcp server
- なんかそういうサービスがあるのかと思ってたけど、これはツールっぽい。管理画面もない。
- いわゆる mcp tool としてあるだけ。これを登録すれば簡単にAIから呼び出せるよって感じ。

setup
```bash
claude mcp add-json aws-mcp --scope user \
  '{"command":"uvx","args":["mcp-proxy-for-aws@latest","https://aws-mcp.us-east-1.api.aws/mcp","--metadata","AWS_REGION=ap-northeast-1"]}'
```

実行してみる
```bash
$ claude
❯ s3バケットをリストアップして

⏺ S3バケットを一覧表示します。

  Called aws-mcp
```
